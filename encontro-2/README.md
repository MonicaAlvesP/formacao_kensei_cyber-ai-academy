# Projeto Kensei - Encontro 2

Este repositório abriga os códigos criados durante o Encontro 2, organizados em pastas para facilitar os estudos.
Todo o projeto foi construído em etapas, utilizando assistentes de Inteligência Artificial.

## Histórico de Prompts Utilizados

Abaixo está a lista exata de comandos (prompts) que utilizei para guiar a IA na construção deste projeto, do início ao fim.

***

### 1. Olá Nome

> "Crie um script python que peça o nome e diga "olá" com o nome maiusculo"

### 2. Conversor de Temperatura

> "crie um script python que converta celsius em fahrenheit. Peça um valor e mostre-o formatado."

### 3. Lista de Compras

> "Crie um programa de lista de compras. Adicionar, visualizar, remover item e sair. Crie usando loop while com menu"
>
> "explique linha por linha do código da #file:lista_compras.py"

### 4. Quiz de Cybersegurança

> "Crie um quizz sobre cybersegurança. 3 opções cada, mostre a pontuação e se passou de 3+ acertos"

### 5. Gerador de Senhas

> "Crie um gerador de senhas. Usuario irá escolher o tamanho e se ele quer maiusculas, numeros, simbolos. Use random"

### 6. Organização, Documentação e Ajustes Finais

> "separe por pastas. E crie documentações para cada arquivo contido dentro da pasta, passo a passo, explicando o código."

***

## Estrutura do Projeto

- `01_ola_nome/` - Script simples de input/output
- `02_conversor_temperatura/` - Cálculos matemáticos com try/except
- `03_lista_compras/` - Uso de Listas e laços While com menus
- `04_quiz_cyber/` - Estruturas de Dicionários e avaliações lógicas
- `05_gerador_senhas/` - Uso prático da importação de pacotes internos (random, string)
